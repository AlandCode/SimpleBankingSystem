# SimpleBankingSystem
ئەم کۆدە داتابەیسێک دروست دەکات کە بە بەکارهێنانی C# ڕێگەت پێدەدات:  ​بەکارهێنەر تۆمار بکەیت لە رێگەی فرەیمۆرکی بە بەکارهێنای   ناو و باڵانسی پارەکەیان پارێزراو بێت.  ​پارە بجووڵێنیت: هەموو جووڵەیەکی دارایی لە خشتەی Transactionsـدا تۆمار دەکات.  ​پەیوەندی: بەهۆی Foreign Keyـەوە، هەموو مامەڵەیەک دەبەسترێتەوە بە یوزەرێکی دیاریکراوەوە.
